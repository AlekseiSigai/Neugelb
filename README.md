# 🎬 Neugelb App

An iOS application that displays movies using the TMDB API.

---

## ✨ Features

* 📃 Latest movies list
* 🔄 Infinite scrolling (pagination)
* 🎬 Movie details screen (overview, images, trailers)
* 🔍 Search with autocomplete suggestions

---

## 🧠 Architecture

The project follows a **MVVM + Router** approach.

---

## 🔐 Configuration & Environment

The project uses `.xcconfig` files for:

* Versioning
* Build configuration
* Environment variables (API keys)

---

## 🚀 Getting Started

### 1. Clone

```bash
git clone https://github.com/AlekseiSigai/Neugelb.git
```

### 2. Run

Open the project in Xcode and run on iPhone simulator or device.
